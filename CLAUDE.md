# Project Rules

## File Organization

- All new HTML files go in `/dashboard` or `/demo-sites` depending on purpose
- All new JavaScript scripts go in `/outreach` unless they are core runners (like `index.js`)
- All new CSV or data files go in `/data`
- All new demo websites for clients go in `/demo-sites/[business-name]/`
- Never create files in the root folder except `index.js` and `README.md`

## Git

- Always do a git commit after any major change
